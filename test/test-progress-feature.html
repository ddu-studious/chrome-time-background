进度功能说明

使用方式

  • 手动设置进度：编辑任务时勾选「启用进度追踪」，输入已完成数量和总数量
  • 自动完成：当进度达到 100% 时，任务会自动标记为已完成
  • 进度颜色：0-30% 橙色、30-60% 紫色、60-100% 绿色
  • 动画效果：进行中的任务会显示条纹动画，已完成的任务动画停止

进度逻辑

  • 进度数据存储为 { current: 3, total: 5 }
  • 百分比 = current / total × 100%
  • 支持随时修改已完成数和总数
  • 不启用进度时，任务只有「完成/未完成」两种状态
