纯百分比模式说明

新版改进

  • 简化交互：移除「步数」模式，只保留纯百分比设置
  • 双向同步：滑块和输入框实时同步，选择更灵活
  • 精确控制：支持直接输入 0-100 的任意数值
  • 修复问题：解决了旧版 90% 自动跳到 100% 的 bug

数据存储

  • 进度数据存储为纯数字 progress: 60（百分比）
  • 旧版 { current: 3, total: 5 } 格式会自动转换
  • 只有 100% 时任务才会自动标记为已完成
  • 支持 0% 到 100% 的任意整数值
