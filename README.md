# 中国风景时钟 Chrome扩展

一个展示时间、日期、天气和精美中国风景的Chrome新标签页扩展。

## 版本

当前版本：1.4.0

## 功能特点

- 显示当前时间和日期（支持12/24小时制切换）
- 显示农历日期
- 显示节假日信息
- 精美的中国风景背景图片轮播（支持自定义背景切换间隔）
- 实时天气信息显示（支持摄氏度/华氏度切换）
- 多语言支持（中文/英文）
- 备忘录功能（支持创建、编辑、删除和分类管理）
- 磨砂玻璃效果的现代界面设计
- 平滑的背景切换动画
- 用户自定义设置

## 安装方法

1. 克隆本仓库到本地
2. 打开Chrome浏览器，进入扩展管理页面（chrome://extensions/）
3. 开启开发者模式
4. 点击"加载已解压的扩展程序"，选择本项目目录

## 使用指南

- 点击设置图标打开设置面板
- 可以自定义时间格式、温度单位、背景切换间隔等
- 点击备忘录图标打开备忘录面板
  - 点击"+"按钮添加新备忘录
  - 点击编辑按钮修改现有备忘录
  - 点击删除按钮删除备忘录
  - 备忘录支持按工作、生活、学习和其他进行分类
- 按空格键可以手动切换背景图片
- 点击刷新按钮可以立即更新天气信息
- 点击语言切换按钮可以切换显示语言

## 技术栈

- JavaScript (ES6+)
- Chrome Extension API
- 和风天气API
