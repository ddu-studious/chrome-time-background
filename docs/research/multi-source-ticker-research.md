### 1.2 候选数据源

#### Reddit — 推荐 ✅

**API**: `https://www.reddit.com/r/{subreddit}.json`

- **认证**: 无需认证，但需设置 User-Agent
- **费用**: 免费
- **限制**: 无认证请求可能被限速，建议设置合理的 User-Agent
- **数据质量**: 优秀，r/programming、r/technology 等子版块有高质量技术内容
- **CORS**: Chrome 扩展通过 host_permissions 可直接访问

**示例响应**:
```json
{
  "data": {
    "children": [
      {
        "data": {
          "title": "文章标题",
          "url": "https://...",
          "score": 1234,
          "subreddit": "programming",
          "num_comments": 56
        }
      }
    ]
  }
}
```

**结论**: 强烈推荐，免费无需认证，数据质量高。

---

#### DEV.to — 推荐 ✅

**API**: `https://dev.to/api/articles`

- **认证**: 无需认证（公开文章）
- **费用**: 完全免费
- **限制**: 无明确限制
- **数据质量**: 优秀，技术社区精选文章
- **CORS**: 原生支持 CORS，Chrome 扩展直接可用

**查询参数**:
```
per_page=10     # 每页数量
top=7           # 最近7天热门
tag=ai          # 按标签筛选
```

**示例响应**:
```json
[
  {
    "title": "文章标题",
    "url": "https://dev.to/...",
    "description": "简短描述...",
    "positive_reactions_count": 234,
    "comments_count": 12,
    "tag_list": ["javascript", "ai"]
  }
]
```

**结论**: 强烈推荐，API 设计优秀，无需认证，天然适合技术用户。

---

#### Product Hunt — 条件推荐 ⚠️

**API**: `https://api.producthunt.com/v2/api/graphql`

- **认证**: 需要 OAuth Client Token（免费注册获取）
- **费用**: 免费
- **限制**: 6250 复杂度点 / 15 分钟
- **数据质量**: 优秀，每日最新产品发布
- **CORS**: 需要认证 Token，建议通过 background.js 代理

**结论**: 需要用户自行注册获取 Token，实现复杂度较高。暂不推荐作为第一批数据源。

---

#### X/Twitter — 不推荐 ❌

- **Official API**: 起价 $100/月（Basic），企业版 $42,000+/月
- **Nitter**: 开源前端，提供 RSS，但公共实例极不稳定，频繁被封
- **XRSS**: 第三方工具，不稳定

**结论**: 成本过高或不稳定，不推荐。

---

#### YouTube — 不推荐 ❌

- **Official API**: 需要 API Key + Google Cloud 项目
- **OpenRSS 转换**: 不稳定，延迟高
- **YouTube Charts RSS**: 仅提供音乐排行，非技术内容

**结论**: 需要 API Key 配置，对普通用户不友好，且内容不匹配技术定位。不推荐。

## 二、UI 优化方案

### 2.1 现有问题

1. Ticker 位置在时间区域上方，但不在最顶部
2. 连续滚动模式信息密度过高，不易阅读
3. 缺少描述信息，无法快速了解内容

### 2.2 优化方案：单条轮播 + 简介

**设计改进**:
- 位置移到页面最顶部（全宽条状栏）
- 改为单条轮播模式，每 5 秒切换一条
- 每条显示：数据源图标 + 标题 + 简短描述 + 指标
- 淡入淡出过渡动画
- 支持多个"频道"数据源

**视觉设计**:
```

│ 🔥 owner/repo-name  ⭐1.2k                                 ↻ ▾ │
│    AI-powered code editor with real-time collaboration         │
└─────────────────────────────────────────────────────────────────┘
```
